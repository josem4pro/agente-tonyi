# 玻尔®️ 科研空间站

本文将帮您快速了解并开始使用 玻尔®️ / Bohrium®️ 平台。

## 玻尔®️ / Bohrium®️ 是什么[​](#玻尔️--bohrium️-是什么 "标题的直接链接")

玻尔®️ / Bohrium®️ （下文统一称为 Bohrium）是深势科技打造的科研云平台。Bohrium 致力于打造团队协作式的科研平台，以赋能微尺度科学研究与工业设计。您可以在 Bohrium 体验 DPA、Uni-Mol、Uni-RNA、Uni-Fold 等科学大模型，并使用这些模型开展科研工作。Bohrium 提供了 Notebook平台、课程平台、比赛平台等教学实训工具，搭载了基于大语言模型的 AI 科研助手，提供了科研APP平台和开发者工具，最大程度地提高创新效能、解放科研生产力。Bohrium 还深度优化了第一性原理计算、分子动力学等微尺度科学计算算法与软件，提供了海量高性能算力与高效便捷的计算模拟环境。

如有任何问题需要咨询，请您联系企业微信 17710231129
