# Open in Bohrium Browser Extension

[Bohrium Notebook](/en/docs/userguide/Notebook) provides a one-stop interactive environment for writing and running code, which allows you to conveniently connect to computing resources and share documents with others. To make it easy for you to run Notebooks from websites like GitHub, you can use the following browser extensions. Once installed, simply click on the extension icon on the target website to open the notebook in Bohrium and start running it.

Click the extension icon on the target page:
![Target Page](/en/assets/images/screenshot-20230612-211048-a24a9ba6c352b41ae1f02db7c61b9c2e.png)

Open the notebook on Bohrium and start running it:
![Open in Bohrium](/en/assets/images/screenshot-20230612-211135-bb66078e4aaa753d3ed8b034ee230db1.png)

Currently, the Open in Bohrium extension supports opening .ipynb files on GitHub.

For more information on using Bohrium Notebook, please click [Bohrium Notebook Help Documentation](/en/docs/userguide/Notebook).

The installation methods for the Open in Bohrium extension are as follows:

* [Chrome Extension Installation](/en/docs/extensions/OpenInBohrium#chrome-extension-installation)
* [Edge Extension Installation](/en/docs/extensions/OpenInBohrium#edge-extension-installation)

## Chrome Extension Installation[​](#chrome-extension-installation "标题的直接链接")

You can search for `Open in Bohrium` in the [Chrome Web Store](https://chrome.google.com/webstore/category/extensions) or directly open the [Open in Bohrium extension page](https://chrome.google.com/webstore/detail/open-in-bohrium/phncjiapfbolllopjlgjfijieidoilla) to install.

![Search Extension](/en/assets/images/screenshot-20230612-211414-efe18e532325f789525d549b1e6fa86b.png)

## Edge Extension Installation[​](#edge-extension-installation "标题的直接链接")

You can open and install the [Open in Bohrium Chrome Extension](https://chrome.google.com/webstore/detail/open-in-bohrium/phncjiapfbolllopjlgjfijieidoilla) directly in the Edge browser.

For more information on using Bohrium Notebook, please click [Bohrium Notebook Help Documentation](/en/docs/userguide/Notebook).
